Foundation Model Benchmark Results

This website summarizes benchmark results for foundation models evaluated across five datasets. Its goal is to present each model's performance in a clear, accessible way using F1 score, accuracy, precision, and ROC-AUC.

For GitHub Pages deployment, keep this file as index.html in the root of the repository or in a docs/ folder. This page will try to load results.json automatically. It also supports uploading a local JSON file for testing.
